###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i> The code declares a variable named greeting using the let keyword. It then assigns an empty object to a variable named greetign. Finally, it logs the value of greetign to the console. The output of this code will be a ReferenceError because greetign is not defined. The correct variable name is greeting, which was declared earlier with the let keyword.</i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The function sum(a, b) takes two arguments a and b and returns their sum. The function is called with arguments 1 and "2". The second argument "2" is a string, not a number. Therefore, when the function tries to add 1 and "2", it concatenates them as strings instead of adding them as numbers. The result of this operation is the string "12".</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>The name parameter is not passed to the function, it is undefined. Therefore, when the function tries to interpolate undefined into the string, it results in "Hi there, undefined".</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B

<i>The forEach() method is called on the nums array with an anonymous function that takes one argument num. The function checks if the value of num is truthy. If it is, the value of count is incremented by 1.
Since only the values of 1, 2, and 3 are truthy in JavaScript, the output of this code snippet is B: 2.</i>

</p>
</details>
